# Data Transfer Reference

## Purpose

This section provides authoritative information about methods used to move data between systems.

It describes:
- different transfer approaches
- their characteristics and limitations
- where each method is appropriate

This section does not provide step-by-step instructions.

## Relationship to storage

Data transfer methods move data between systems.

They do not:
- provide authoritative storage
- manage long-term data persistence
- replace storage services

For storage systems and comparison, see:
- [Storage comparison](../storage/storage-comparison.md)
- [Storage reference](../storage/index.md)
