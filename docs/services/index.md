# Services

## UCT eResearch services

UCT provides services to support different parts of your research workflow, including:

- computing (HPC)
- data storage
- data transfer
- research software and collaboration tools

## How services fit together

Research workflows often involve multiple services:

- GitLab → for code and collaboration  
- Storage → for data  
- Transfer tools → to move data  
- HPC → for compute  

Understanding how these fit together will help you work more efficiently.
