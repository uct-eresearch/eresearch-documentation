# Research Software

## What this service is

Research software provides tools and platforms for developing, managing, and sharing code used in research.

This includes:

- version-controlled repositories
- collaborative development environments
- platforms for sharing and maintaining research code

---

## When to use this service

Use research software services when you need to:

- manage code for a research project
- collaborate with others on software or analysis workflows
- track changes to code over time
- organise and share scripts, notebooks, or software components

---

## What this service provides

Research software services at UCT include:

- repository hosting for version-controlled code
- access to collaborative development platforms
- support for managing and sharing research software

These services enable structured and reproducible development of research code.
