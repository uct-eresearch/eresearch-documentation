# Choose the right service

## When to use this

You are not sure which UCT service fits your work.

---

## What you need to decide first

1. **What are you trying to do?**
   - compute-intensive analysis → HPC  
   - data storage → Research data storage  
   - data transfer → Data transfer  
   - code collaboration → GitLab / research software  

2. **How large is your data or workload?**
   - small → local tools may be sufficient  
   - large → use institutional services  

3. **Do you need shared or managed infrastructure?**
   - yes → use UCT services  
   - no → local workflow may be sufficient  

---

## Recommended path

- Running large analyses  
  → Task: Run large-scale analysis  

- Storing or sharing data  
  → Task: Store and share research data  

- Moving data  
  → Task: Move data securely  

- Collaborating on code  
  → Task: Collaborate on code  

---

## Common mistakes

- using HPC for storage instead of computation  
- storing large data locally without backup  
- choosing tools before defining the problem  

---

## Next steps

- select a task that matches your goal  
- follow the recommended path from that task
