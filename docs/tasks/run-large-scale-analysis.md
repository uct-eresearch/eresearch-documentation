# Run large-scale analysis

## When to use this

You want to:

- run analyses that take a long time (hours to days)
- process large datasets
- scale work across multiple cores or nodes
- use GPUs for compute-intensive workloads

---

## What this involves

Running large-scale analysis at UCT typically uses:

- High Performance Computing (HPC) for compute
- Research data storage for input and output data
- Data transfer tools to move data to and from HPC
